# Requirements

- Ansible core (version >= 2.14)
  - Python3 and Pip
- Packer (version >= 1.8)
  - Packer builder arm
- Terraform (version >= 1.3)

# Ansible

`cd ansible`

# Packer

`cd packer`

# Terraform

`cd terraform`
